== Description ==
News Headline Ticker is best wordpress plugin to display breaking news or news ticker at your website!

Plugin Documentation: [https://wordpress.org/plugins/rhx-news-ticker/](https://wordpress.org/plugins/rhx-news-ticker/)

*   [Need Help?](https://www.facebook.com/rihan.zihad/)		
*   [About Author](https://www.facebook.com/rihan.zihad/)


== You can choose between 3 effect ==
*   Typing – Display your ticker like typing style.
*   Slide – Display your ticker like sliding style.
*   Fade in/out – Display your ticker like fade in/out style.

== You can select/change ==
*   Post Category.
*   Effect Type.
*   Number of Post.
*   Post Display Order.
*   Background Color.
*   Text Color.
*   Text Hover Color.
*   All are dynamic.

= Usage =

* Go to your Dashboard after installation and navigate to "Settings >> Ticker Settings" to configure the news ticker.

= Features =

  * Very easy installation
  * Shortcode system
  * Unlimited color variation by using jQuery color picker.
  * Responsive design
  * Custom post supported
  * Flexible and easy to use
  * Lightweight JQuery


== Installation ==
1. Upload `rhx-news-ticker` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Copy and paste <strong>[rhxnews-ticker]</strong> in the post/page" to display news ticker.
